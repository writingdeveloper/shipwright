# UI i18n (next-intl) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:executing-plans. Steps use `- [ ]`.

**Goal:** Internationalize the starter with next-intl + URL-prefix routing (`as-needed`), SEO hreflang, and a `@repo/i18n` package + en/ko demo, keeping default-locale URLs and the 40 e2e unchanged.

**Architecture:** `@repo/i18n` owns the mechanism (routing, request config, locale-aware navigation); `apps/web/messages/` owns the strings. All routes move under `app/[locale]/`; `proxy.ts` composes next-intl middleware with the existing nonce-CSP + auth rate-limit. `@repo/seo` emits hreflang.

**Tech Stack:** next-intl, Next 16 App Router, Turborepo, Playwright, Vitest.

Spec: `docs/superpowers/specs/2026-06-20-i18n-design.md`.
